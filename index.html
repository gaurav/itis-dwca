itis-dwca

A DwC-A version of the ITIS taxonomic database, by permission of the ITIS Data Specialized. I download the database dump from the ITIS website, then use dwca-hunter by the Global Names Architecture to generate a DwC-A file from ITIS.

If everything's working correctly, you should get the latest DwC-A files from:

  • http://gaurav.github.com/itis-dwca/latest/version.txt: the version information for the latest version.
  • http://gaurav.github.com/itis-dwca/latest/dwca.tar.gz: the DwC-A file for the latest version.
  • http://gaurav.github.com/itis-dwca/latest/taxa.txt: A CSV file containing a list of all the taxa from the DwC-A file.

Please note that all data used from the DwC-A file should be credited to ITIS as follows: Retrieved [month, day, year], from the Integrated Taxonomic Information System (ITIS) (http://www.itis.gov). The month/day/year can be available from the first line of the version.txt file.

Please report any issues you notice with this distribution, and use at your own risk!
