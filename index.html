itis-dwca

A DwC-A version of the ITIS taxonomic database, by permission of ITIS. I download the database dump from the ITIS website, then use dwca-hunter by the Global Names Architecture to generate a DwC-A file from ITIS.

If everything's working correctly, you should get the latest DwC-A files from:

  • http://gaurav.github.com/itis-dwca/latest/version.txt: the version information for the latest version (created on July 05, 2018).
  • http://gaurav.github.com/itis-dwca/latest/dwca.tar.gz: the DwC-A file for the latest version (based on the ITIS database published on July 02, 2018).
  • http://gaurav.github.com/itis-dwca/latest/taxa.txt.gz: A Gzipped CSV file containing a list of all the taxa from the DwC-A file.

These files were created as part of the Taxosaurus TNRS service developed during the Phylotastic hackathon at National Evolutionary Synthesis Center, Durham NC, USA.

Citation

Please note that all data used from the DwC-A file should be credited to ITIS as follows:
